# Pomodoro App

A modern Pomodoro timer built with Python and Tkinter. Track your work and break sessions efficiently with a customizable timer and notifications.

## Features

- Set your own work and break durations
- Visual countdown with animated cup icon
- Customizable themes and colors
- Sound notifications for session start/end
- Easy-to-use GUI with start, pause, and reset
- Optional multiple timers for focus sessions
